Guide for installing a **World of Warcraft private server** using **AzerothCore 3.3.5** with the **Playerbots module** running in Docker.

Guide d'installation d'un **serveur privé World of Warcraft** basé sur **AzerothCore 3.3.5** avec le **module Playerbots** exécuté via Docker.

---

# 🇬🇧 English

## Requirements

Recommended server configuration:

- Ubuntu Server 24.04 LTS  
- 8 GB RAM  
- 2–4 vCPU  
- 60 GB storage  

## Installation

The complete installation guide is available here:

👉 **[Installation Guide](installation_guide.ssh)**

This guide explains how to:

- install Docker
- deploy AzerothCore
- configure the realm
- create game accounts
